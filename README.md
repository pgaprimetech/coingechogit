This is a Python script used to scrape cryptocurrency data (Coin Name,
 Price (USD), 1h %, 24h %, 7d %, 24h Volume (USD), Market Cap (USD), and Coin Link) from all the
 pages of https://www.coingecko.com, and save all the results as an Excel file.
The script also automatically sends the Excel file to your email after scraping all the data.


Add your SMTP credentials to the .env file.

install
python version:3.10

install playwright browsers:
playwright install

install recquired  moduls using this command:
pip install -r  requirements.txt

Then start the scrapper by running this command:
python crypto_scraper.py

Send result from the output manually to your email run this command:
python email_sender.py
