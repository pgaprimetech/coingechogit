# coingechogit
This is a Python script used to scrape cryptocurrency data (Coin Name,  Price (USD), 1h %, 24h %, 7d %, 24h Volume (USD), Market Cap (USD), and Coin Link) from all the  pages of https://www.coingecko.com, and save all the results as an Excel file. The script also automatically sends the Excel file to your email after scraping all the data.
